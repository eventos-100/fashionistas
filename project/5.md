#### 1. Theme Provider Conflicts
```typescript

// Solution: Ensure single ThemeProvider at root

const theme = createTheme({
  // Your theme config

  <App />
```

```typescript

// Solution: Use NoSsr wrapper for client-only components
import NoSsr from '@mui/material/NoSsr';

<NoSsr>
  <ClientOnlyComponent />
</NoSsr>
```
